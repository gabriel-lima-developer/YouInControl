# ADR-0005 - Padrao de formatacao e chaves em C#

## Status

Aceito

## Contexto

O projeto YouInControl e um projeto pessoal, laboratorio tecnico e portfolio. Por isso, precisa de consistencia visual para facilitar leitura, manutencao, revisoes e contribuicoes futuras.

## Decisao

Usar chaves de abertura na mesma linha da declaracao, no estilo K&R, em vez do padrao Allman tradicional do C#.

## Consequencias

- Consistencia visual em todo o codigo C#.
- Menor retrabalho em revisoes e alteracoes futuras.
- Alinhamento com a preferencia do mantenedor.
- Orientacao clara para agentes de IA e futuras contribuicoes.
